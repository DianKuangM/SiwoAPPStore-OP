# SiwoAPPStore-OP
拾物应用商店 部分源码开源项目
# 项目初衷
本项目旨在分享拾物应用商店的前端界面与基础交互逻辑。为确保核心业务逻辑的安全与独特性，本次开源仅包含以下 4 个文件：

 index.php  (首页逻辑)

 login.php  (登录模块)

 register.php  (注册模块)

 style.html  (主样式与结构)

其余核心功能（如后端 API、数据库结构、管理后台等）暂未开放。开发者可基于此前端框架，结合自身需求进行二次开发
演示站：https://oldchat.shop
